# GOAL

The objective was to make a single page application that contains the Tip Tap game, where it is possible to choose the player to choose and show the winning player, if there is one, and not indicate that there was no winner.

# TECHNICAL DESCRIPTION

This exercise was developed using HTML, CSS, and Angular
